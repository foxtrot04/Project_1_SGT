# scroll-toggle-site

## Technology used in the Project:
- Vue.js 2
- GSAP
- SASS
- Vue CLI

## Dependencies used in the Project:
- gsap
- sass
- sass-loader

## Download Dependencies
```
npm install gsap

npm install sass sass-loader --save-dev
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

## Website will be available at:
```
http://localhost:8080/
```
